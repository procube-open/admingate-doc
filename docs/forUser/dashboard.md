## サイドバー

ホームページからアクセス可能なページに、別ページからでもアクセスを可能とするための機能です。


## 言語選択

クリックするとメニューが開き、日本語と英語が選択することができます。

## モード選択

クリックするとメニューが開き、作業者モードと管理者モードを選択することができます。ユーザによってモードを切り替えることでそれぞれアクセス可能なページが異なります。
詳細は[こちら](/docs/forUser/work)を参照して下さい。

## フィルタ

作業一覧、接続先一覧、履歴一覧に対してはフィルタ機能が用意されています。
メニューに表示されているボタンを押すことで各種絞り込みを適用できます。

### 文字列検索
それぞれ以下のパラメータに対して文字列検索がかかります。
- 作業一覧
  - 作業名
  - 作業ID
  - 接続先ホスト名
  - 接続先補足
- 接続先一覧
  - ID
  - ホスト名
  - 補足
- 履歴一覧
  - ホスト名
  - ユーザ名

ただし、スペースなどによる複数文字列検索には対応していません。

### お知らせ
全ユーザに対して告知されるお知らせの一覧が表示されています。

### フィルタ条件保存
保存したいフィルタ条件の状態で右上のプラスボタンを押すことで、そのフィルタ条件をブラウザに保存することができます。
以降は保存されたフィルタボタンをクリックすることで、そのフィルタを即座に適用させることができます。

フィルタ条件をクリックすることで右上がマイナスボタンになるので、これを押すことでフィルタを削除できる。
