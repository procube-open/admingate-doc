# WebUI概要
ここではWebUIの概要について説明します。

## ホームページ
AdminGateのWebUIにアクセスすると、まずホームページが開きます。
ホームページはユーザへのお知らせの確認や、従事作業の選択など、各種ページへのアクセスハブの役割を持ちます。

## サイドバー
どのページからでも開閉可能なサイドバーが存在します。
サイドバーを開くことで、ホームページからアクセス可能なページに、別ページからいつでもアクセスすることができます。

## 言語選択

言語は日本語と英語に対応しています。
ヘッダーの言語欄をクリックすることでメニューが開き、日本語と英語を選択することができます。

## モード選択

ユーザは作業者モードと管理者モードを選択することができ、初期状態では作業者モードとなっています。
作業者と管理者の詳細は[こちら](/docs/forUser/work)を参照して下さい。

ヘッダーのユーザアイコンをクリックすることでメニューが開き、作業者モードと管理者モードを選択することができます。

モードを切り替えることでそれぞれアクセス可能なページがユーザによって異なります。

## フィルタ

作業一覧、接続先一覧、履歴一覧に対してはフィルタ機能が用意されています。
メニューに表示されているボタンを押すことで各種絞り込みを適用できます。

### 文字列検索
フィルタの一つとして、文字列検索を行うことができます。
検索窓に文字を入力することで、それぞれ以下のパラメータに対して文字列の部分一致検索がかかります。

- 作業一覧
  - 作業名
  - 作業ID
  - 接続先ホスト名
  - 接続先補足
- 接続先一覧
  - ID
  - ホスト名
  - 補足
- 履歴一覧
  - ホスト名
  - ユーザ名

ただし、スペースなどによる複数文字列検索には対応していません。

### お知らせ
全ユーザに対して告知されるお知らせの一覧が表示されています。

### フィルタ条件保存
フィルタ条件を保存し、保存したフィルタ条件を即座に呼び出すことができます。

保存したいフィルタ条件の状態で右上のプラスボタンを押すことで、そのフィルタ条件をブラウザに保存することができます。
以降は保存されたフィルタボタンをクリックすることで、そのフィルタを即座に適用させることができます。

フィルタ条件をクリックすることで右上がマイナスボタンになるので、これを押すことでフィルタを削除することができます。
